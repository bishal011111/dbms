Frequently Asked Questions

Create a Gmail Account

  1. Go to Gmail.
  2. Click on "Create account" and select "For myself" or "For my business."
  3. Fill in your first and last name.
  4. Choose a username (your email address) and a password.
  5. Confirm your password.
  6. Click "Next."
  7. Provide your phone number for account verification.
  8. Enter the verification code sent to your phone.
  9. Fill in the required personal information.
  10. Review and accept Google's Terms of Service and Privacy Policy.
  11. Click "Next" to complete the account setup.

Create a GitHub Account

  1. Go to GitHub.
  2. Click on "Sign up" in the upper right corner.
  3. Enter your desired username, email address, and password.
  4. Complete the puzzle to verify you are human.
  5. Click "Create account."
  6. Choose your plan (free or paid).
  7. Verify your email address by clicking the link sent to your email.

Connect VS Code to GitHub

  1. Install Visual Studio Code.
  2. Open VS Code and go to the Extensions view by clicking on the square icon in the sidebar or pressing Ctrl+Shift+X.
  3. Search for "GitHub" and install the "GitHub Pull Requests and Issues" extension.
  4. Press Ctrl+Shift+P to open the Command Palette, and type "Sign in to GitHub" and select it.
  5. Follow the prompts to log in to your GitHub account.

Clone/Pull Repository from GitHub

  1. Open your browser and go to the GitHub repository you want to clone.
  2. Click on the "Code" button and copy the repository URL.
  3. Open VS Code.
  4. Press Ctrl+Shift+P to open the Command Palette, type "Git: Clone" and select it.
  5. Paste the repository URL and choose a local directory to save the cloned repository.
  6. VS Code will clone the repository and open it.

Push Repository to GitHub

  1. Open your project folder in VS Code.
  2. Open the Source Control view by clicking on the branch icon in the sidebar.
  3. Stage your changes by clicking the "+" icon next to the files you modified.
  4. Write a commit message in the text box and click the checkmark icon to commit the changes.
  5. Click on the ellipsis (...) in the Source Control view and select "Push" to push your changes to GitHub.

Publish Repo as Live Site

  1. Go to your repository on GitHub.
  2. Click on the "Settings" tab.
  3. Scroll down to the "GitHub Pages" section.
  4. Under "Source," select the branch you want to publish (usually main or gh-pages).
  5. If you want to publish from a specific folder (e.g., docs), set the folder in the "Source" dropdown.
  6. Click "Save."
