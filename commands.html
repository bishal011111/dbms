SQL Commands

1. CREATE

Description: The CREATE command is used to create a new database object, such as a table, view, index, or stored procedure.

CREATE TABLE Employees (
    EmployeeID int,
    FirstName varchar(255),
    LastName varchar(255),
    BirthDate date,
    Position varchar(255)
);

This example creates a new table named Employees.

2. DROP

Description: The DROP command is used to delete a database object permanently, such as a table, view, index, or database.

DROP TABLE Employees;

This example deletes the Employees table and all its data.

3. ALTER

Description: The ALTER command is used to modify an existing database object, such as a table, by adding, deleting, or modifying columns and constraints.

ALTER TABLE Employees
ADD Email varchar(255);

This example adds a new column named Email to the Employees table.

4. ADD

Description: The ADD command is typically used with the ALTER command to add a new column or constraint to an existing table.

ALTER TABLE Employees
ADD PhoneNumber varchar(20);

This example adds a new column named PhoneNumber to the Employees table.

5. UPDATE

Description: The UPDATE command is used to modify existing records in a table.

UPDATE Employees
SET Position = 'Manager'
WHERE EmployeeID = 1;

This example updates the Position of the employee with EmployeeID 1 to 'Manager'.

6. SET

Description: The SET command is used in the UPDATE statement to specify the column that should be modified and the value it should be set to.

UPDATE Employees
SET LastName = 'Smith'
WHERE EmployeeID = 2;

This example sets the LastName of the employee with EmployeeID 2 to 'Smith'.

7. INSERT

Description: The INSERT command is used to add new records into a table.

INSERT INTO Employees (EmployeeID, FirstName, LastName, BirthDate, Position)
VALUES (3, 'Jane', 'Doe', '1990-05-15', 'Developer');

This example inserts a new record into the Employees table.

8. SELECT

Description: The SELECT command is used to retrieve data from one or more tables.

SELECT FirstName, LastName
FROM Employees
WHERE Position = 'Developer';

This example retrieves the FirstName and LastName of all employees who are developers.

These commands form the basic building blocks of SQL and are essential for creating, modifying, and querying database objects and data.
